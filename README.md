# 🌿 Bug Read & Play - 곤충 영어 리딩 플랫폼

> **"좋아하는 곤충을 관찰하며 자연스럽게 익히는 어린이 영어 독후활동 플랫폼"**

**Bug Read & Play**는 아이가 곤충을 공부(과학적 탐구)하는 즐거움을 느끼는 과정에서 자연스럽게 영어를 읽고(Reading), 익히고(Vocabulary), 확인하고(Quiz), 표현하는(Canvas Drawing/Writing) 7단계 순환 학습 웹 애플리케이션입니다.

---

## ✨ 주요 기능 및 특징

1. **📖 12종 곤충 동화 스토리북 (4단계 레벨 제공)**
   * 무당벌레, 개미, 나비, 꿀벌, 장수풍뎅이, 잠자리 등 총 12종의 생태 동화 제공.
   * 유치원생(Lv.1)부터 초등학교 고학년(Lv.4)까지 연령별 난이도 맞춤형 6페이지 스토리북 구성.

2. **🔊 오디오 낭독 및 실시간 단어 추적 하이라이트**
   * Web Speech API(TTS) 기반 원어민 영어 오디오 지원.
   * **현재 음성으로 낭독 중인 영어 단어를 실시간으로 노란색 하이라이트 및 확대**하여 자녀의 시선 매칭을 돕는 첨단 독서 기능.

3. **🔤 3D 카드 단어 학습**
   * 단어 카드를 클릭하면 3D 회전 애니메이션과 함께 한글 뜻과 영어 예문 제공.
   * 단어와 예문 개별 듣기 지원 및 학습 진척도 자동 체크.

4. **🎯 정답 피드백 퀴즈**
   * 책을 잘 이해했는지 확인하는 정답 인터랙션 퀴즈 게임. 정답률 평균을 학부모 대시보드에 연계.

5. **🎨 독후 활동 (HTML5 Canvas 스케치북 & 글쓰기)**
   * 태블릿(터치) 및 PC(마우스) 드로잉을 모두 지원하는 드로잉 캔버스 내장.
   * 추천 예시 문장을 활용해 영어 문장을 직접 쓰고, 원어민 발음으로 들어보는 자가 점검 글쓰기 노트.
   * 아이가 그린 소중한 그림과 글은 브라우저에 이미지 데이터로 영구 보관.

6. **🤖 실시간 AI 곤충 친구 (Gemini 1.5 Flash 연동)**
   * 학부모 설정에서 Google Gemini API Key를 등록하면, 아이가 한국어/영어 등으로 묻는 곤충 질문에 AI 박사님이 이모티콘을 섞어 친절하게 대답해 주고 즉석 영어 동화를 생성해 줍니다. (미등록 시 로컬 모의 AI로 작동)

7. **👨‍👩‍👧 학부모 대시보드 (보호자 게이트 포함)**
   * 아이가 함부로 설정을 만지지 못하도록 **랜덤 구구단 연산 인증 모달**을 통과해야만 대시보드 진입 가능.
   * 누적 획득한 별(⭐), 완독권수, 단어 학습 현황 및 **아이가 그린 스케치북 그림 갤러리 보관소** 제공.

## 🛠️ 시작하기 (How to Run)

본 프로젝트는 Node.js 기반 프론트엔드 프로젝트입니다.

### 1. 필수 환경 설치
먼저 컴퓨터에 [Node.js](https://nodejs.org)가 설치되어 있어야 합니다 (LTS 버전 추천).

### 2. 패키지 의존성 설치
프로젝트 루트 폴더에서 터미널을 열고 다음 명령을 실행하여 Vite 빌드 환경을 설정합니다.
```bash
npm install
```

### 3. 개발 서버 구동
로컬 개발 서버를 실행합니다.
```bash
npm run dev
```
실행 후 터미널에 나타나는 주소(기본값: `http://localhost:5173`)로 브라우저를 열면 앱이 구동됩니다.

### 4. 상용 빌드
웹 호스팅(GitHub Pages 등)에 배포하기 위해 빌드 파일을 추출하려면 아래 명령을 사용합니다. `dist` 폴더 안에 최적화된 결과물이 생성됩니다.
```bash
npm run build
```

## 📂 폴더 구조 (Folder Structure)

```
├── index.html              # 메인 SPA HTML 진입 파일
├── package.json            # Vite 빌드 설정 및 스크립트
├── requirements.txt        # 깃허브 배포를 위한 시스템 요구 사양서
├── src
│   ├── main.js             # 상태 관리, 라우터, 전체 코어 연동 로직
│   ├── style.css           # 글로벌 테마, Glassmorphism, 3D 플립 애니메이션 CSS
│   ├── data
│   │   └── insects.json    # 12종 곤충 책 스토리 및 단어/퀴즈 데이터베이스
│   └── components
│       ├── DrawingBoard.js # HTML5 Canvas 드로잉판 및 쓰기 노트 컴포넌트
│       ├── AIChat.js       # 모의 AI 대화창 및 Gemini API 연동 모듈
│       └── ParentDashboard.js # 학부모 대시보드, 갤러리, 환경설정 컴포넌트
```
