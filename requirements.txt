# 1. 필수 시스템 환경 (Required Environments)
# ------------------------------------------------------------------------------
# - Node.js v18.0.0 이상 설치 필요 (다운로드: https://nodejs.org)
# - npm v9.0.0 이상 (Node.js 설치 시 자동 설치됨)

# 2. 사용된 프론트엔드 라이브러리 및 도구 (Key Tech Stacks)
# ------------------------------------------------------------------------------
# - 빌드 도구 (Build Tool): Vite v5.0.0 이상
# - 스타일링 (Styling): HTML5 Canvas API, CSS3 Grid/Flexbox
# - 인공지능 (AI API): Google Gemini API (Web fetch integration)
# - 음성 출력 (Audio): Web Speech API (SpeechSynthesis)

# 3. GitHub 다운로드 후 터미널 실행 방법 (How to Run)
# ------------------------------------------------------------------------------
# 1) 패키지 의존성 설치:
#    npm install
#
# 2) 로컬 개발 서버 실행:
#    npm run dev
#
# 3) 상용 서비스 배포용 빌드:
#    npm run build
